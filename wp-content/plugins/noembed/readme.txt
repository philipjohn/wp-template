Add support for Noembed.com-supported embeds

== Description ==

Noembed provides embeddable content from a large list of sites, even sites without oEmbed support! The Noembed plugin for WordPress lets you embed content from any Noembed-supported site. The list of supported sites is updated daily from Noembed.com.

== Installation ==

 * Uploads and install this plugin as normal, and activate it from the Plugins menu in WordPress.
 * Insert a URL from one of the supported sites into your post content and then view the post.
 * Bam! The content will be embedded.
